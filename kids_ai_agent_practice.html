🛠️ 打造专属 AI 智能体：全流程开发手册

第一部分：基础环境安装 (构建骨架)

1. 安装 Ollama (AI 引擎)：访问 ollama.com 下载。安装后，它是你电脑里运行 AI 的“发动机”。

2. 启动大脑：打开电脑的终端（CMD/Terminal），输入以下命令下载并运行 Hermes 3：

ollama run hermes3

此时，你的电脑已经具备了运行智能体的大脑基础。

3. 安装 AnythingLLM (可视化控制台)：访问 useanything.com 下载桌面版。这是你用来设置 AI “灵魂”和“知识库”的图形化工具。

4. 绑定引擎：打开 AnythingLLM，在设置中选择 Ollama 作为 LLM Provider，它会自动识别到你刚才安装的 Hermes 3。

第二部分：灵魂与记忆配置 (智能体塑造)

1. 创建智能体 (Workspace)：点击左下角 + New Workspace，命名为“课表小管家”。

2. 喂入记忆 (Knowledge Base)：

  • 点击左侧“文档”图标。
  • 上传你的《课表.txt》或作业要求文件。
  • 点击 Embed（向量化），将这些文件转为 AI 可读取的“记忆”。

3. 注入灵魂 (System Prompt 设置)：进入该 Workspace 的设置页面，找到 System Prompt 输入框，填入以下详细模板：

# Role: 你是五年级学生的“课表小助手”。 # Objective: 确保主人准时完成作业和运动。 # Constraint (灵魂禁区): 1. 绝不代写作业！遇到作业题，只允许提供思路，严禁给出答案。 2. 只根据上传的“知识库”内容回答。不知道的内容请说：“我还没学到，请查阅书本”。 # Examples (Few-Shot 示例): - 用户：课表里周三有什么课？ - Agent：周三有科学和体育！记得体育课要穿运动鞋哦，体育老师很看重这个。 - 用户：帮我写一下数学题吧。 - Agent：那我可不能帮，但我可以给你一个类似的计算公式，你自己试试看能不能解开？
给家长的调教建议： 配置完成后，先测试“灵魂”是否稳定。如果它回答得太啰嗦，修改 Prompt 中的 Action (沟通风格) 部分，添加“语气要简短、高效”这样的指令。
